How it works

Find each other here. Settle privately via Idios on Beam.

01 FOR CLIENTS

Post a job or find a worker

Browse workers by skill and reputation. Post a job listing so workers can find you. Agree on terms off-chain via Telegram or Discord.

02 BOTH SIDES

Lock funds via Idios

Client locks payment. Worker locks collateral. Both sides have money at risk before work starts. Install the Idios dapp in Beam Desktop to create the contract.

03 SETTLE PRIVATELY

Deliver and claim

Worker delivers. Client approves or disputes. Settlement happens privately on Beam MimbleWimble. Amounts and parties hidden by default. No platform takes a cut.

New to Idios? Install the dapp in your Beam Desktop wallet to create private escrow contracts.
